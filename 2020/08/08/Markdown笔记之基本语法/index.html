​ 自 用 的 Markdown 笔 记

用的是0.9.92（beta）的typora，与MarkdownPad2有出入

TEST # TEST

TEST1 ## TEST1

TEST2 ### TEST2

TEST3 #### TEST3

[TOC] #目录 //因本主题自动生目录放置左侧边栏，故不显示效果

TEST #Crtl Shift `

TESR #Ctrl B 或者 *TEST*或者_TEST_

TEST #Ctrl I 或者 *TEST* 或者_\TEST___

TEST #Crtl B I 或者***TEST***或者_\_TEST___

TEST #Crtl U 或者 ++TEST++

TEST #Shift Alt 5 或者 ~~TEST~~

TEST #Ctrl K （可带标注）或者 [TEST][1] 回车 [1]:http://www.baidu.com //要带上http(s)协议、 没有标注、1可换成单词

<www.baidu.com> # < www.baidu.com >

TEST1 #>TEST1

TEST2 #>>TEST2

test test1
|TEST|TEST1| 回车 #生成对应1x2无框表格 | | | 回车 #可快速插入空白1x2表格

在其他MarkdownPad2编辑器中：

TEST|TEST1
-| #可生成一个无框1x2表格

TEST|TEST1
-|
TEST2|TEST3 #可生成一个无框2x2表格

-– 或者 *** #至少3个-或者*才能生成单行分隔符

$f(x)=x$ #$f(x)=x$

#Ctrk Shift I 或者![七院](https://cdn.jsdelivr.net/gh/Higgins995/The-Photos-of-Blog//08%E9%99%A2%E5%BE%BD.png “七院” )

  • TEST #+ TEST或- TEST 或 * TEST
    • TEST1 #回车 后按Tab （前提要有一级序号）
  1. TEST #1. TEST
  • TEST #- [x] TEST

TEST^1 

#TEST\[^1]  回车 \[^1]:脚注

TEST^TEST1^ #TEST^TEST1^

TESTTEST2 #TEST~TEST2~

==TEST== #==TEST== (语法高亮)

:city_sunrise: #:citi_sunrise: //输入:+图标的英文单词,注意图标不能插空
